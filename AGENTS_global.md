## python packages
- Python Package Manager가 필요하다면 venv 말고 uv를 사용해
- Python Package 설치가 필요하면 uv init 하고 uv add로 추가해

## git 
- git commit을 수행할 때는 한글로 간단히 작성해
- worktree 작업이 필요하다면 해당 프로젝트의 .worktrees/<worktree-name> 형태로 저장소를 만들어

## 작업요청
- 신규 작업이 요청되면 git pull 을 한 뒤에 코드 업데이트에 문제 없는 지 확인한 뒤 worktree를 만들고 그 안에서 작업해
